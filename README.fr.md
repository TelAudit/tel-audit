# TELAUDIT

**Outil local de documentation et d'audit IPBX pour Windows.**

TELAUDIT analyse des sauvegardes d'IPBX et génère automatiquement de la documentation professionnelle, des rapports d'audit et des schémas.

Le logiciel est conçu pour rester simple, rapide, local et respectueux de la vie privée.

> Version actuelle : **v2.3 beta 3**  
> Contact : **telauditwin@gmail.com**  
> Soutenir le projet : https://ko-fi.com/telaudit

[English](README.md)

---

## Ce que fait TELAUDIT

TELAUDIT peut générer :

- rapports PDF ;
- rapports HTML ;
- exports CSV ;
- audit automatique ;
- schémas IPBX ;
- exports Draw.io / diagrams.net ;
- exports PNG des schémas.

La bêta publique actuelle se concentre sur les sauvegardes de type **Xorcom / Ombutel**. D'autres parseurs pourront être ajoutés progressivement.

---

## Confidentialité

TELAUDIT fonctionne localement sous Windows.

- Aucune télémétrie
- Aucun tracking
- Aucune publicité
- Aucun compte en ligne requis
- Aucune collecte de données
- Aucune donnée IPBX envoyée sur Internet

Vos sauvegardes restent sur votre ordinateur.

## Utilisation rapide

1. Lancez TELAUDIT.
2. Choisissez la langue de l'interface.
3. Glissez-déposez une sauvegarde IPBX ou sélectionnez-la manuellement.
4. Choisissez un dossier de sortie.
5. Cliquez sur **Analyser et générer**.
6. Ouvrez les fichiers PDF, HTML, CSV et schémas générés.

Formats actuellement acceptés :

- `.tar`
- `.tar.gz`
- `.tgz`
- `.gz`

---

## Fichiers générés

Selon le contenu de la sauvegarde, TELAUDIT peut générer :

- rapport PDF principal ;
- rapport HTML autonome ;
- exports CSV ;
- export CSV d'audit ;
- schéma général en PNG ;
- schéma Draw.io modifiable ;
- exports dédiés aux renvois d'appels détectés.
